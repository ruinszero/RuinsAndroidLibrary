# RuinsAndroidLibrary
一个自用Android框架，封装常用库，方便开发。
## Gradle
Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
Add the dependency:
```
dependencies {
	        compile 'com.github.ruinszero:RuinsAndroidLibrary:v0.0.4'
	}
```
