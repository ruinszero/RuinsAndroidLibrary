# RuinsAndroidLibrary
一个Android框架封装库，最终目的是成为一个模块化、插件式的框架。
## Gradle
Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
Add the dependency:
```
dependencies {
	        compile 'com.github.ruinszero:RuinsAndroidLibrary:v0.0.2'
	}
```
